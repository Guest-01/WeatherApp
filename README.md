# tkinter-weatherapp
Cross-platform GUI app that uses API from https://openweathermap.org/
## Installation
Install dependencies
```
pip install -r requirements.txt
```
Put Your API Key (WeatherApp.py)
```Python
import tkinter as tk
import requests

HEIGHT = 500
WIDTH = 600
API_KEY = ''  #Your API_KEY
...
```
